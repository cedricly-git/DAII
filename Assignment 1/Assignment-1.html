Creation of the new datasets

data_income <- data1 %>%
  select(age, visits, income) #creation of the data set data_income by selecting the variables "age", "visits" and "income"
data_gender <- data1 %>%
  select(age, visits, gender) #creation of the data set data_gender by selecting the variables "age", "visits" and "gender"

Adding a new column (variable) to the data_income

median_income <- median(data_income$income) #compute the median of income

data_income <- data_income %>%
  mutate(income_category = factor(ifelse(income > median_income, 
                                         "Above Median Income", 
                                         "Below Median Income"))) #create a new column (variable : income_category) with mutate with the values "Above Median Income" and "Below Median Income"

Creating plot

ggplot(data_income, aes(x = age, y = visits)) +
  geom_point(color = 1) +
  facet_wrap(~(income_category), scale = "free") +
  geom_smooth(method = "lm", se = FALSE) +
  labs(title = "Correlation between Age and Visits by Income Class", 
       x = "Age", 
       y = "Number of Visits") +
  theme_minimal() #generating the graphs for the data set "data_income" with ggplot2
## `geom_smooth()` using formula = 'y ~ x'

Interpretation: Basing the Analysis on Income Class We have a positive correlation, as age increases, doctor visits also increase for both income groups. Below median income class has a steeper slope meaning lower-income individuals seem to visit the doctor more frequently as they age, possibly due having more stress and working more dangerous or strenuous job or having less access to preventative care or wellness. Above Median Income shows a milder increase â Wealthier individuals might have better healthcare access, allowing for preventive care instead of reactive doctor visits. Possible Interpretation: Older individuals in the lower-income group may rely more on reactive (rather than preventive) medical care. Wealthier individuals might have better insurance, private doctors, or healthier lifestyles, reducing their need for frequent visits.
